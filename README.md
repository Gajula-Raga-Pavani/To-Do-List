# 📝 To-Do List Application

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript. This project helps users efficiently manage their daily tasks by adding, completing, and deleting tasks through an intuitive user interface.

## 🚀 Features

- Add new tasks.
- Mark tasks as completed.
- Delete tasks.
- User-friendly and responsive interface.
- Real-time task management.
- Lightweight and easy to use.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
To-Do-List/
│
├── index.html

└── README.md
```

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd To-Do-List
```

3. Open the `index.html` file in your browser.

OR

- Right-click on `index.html`.
- Select **Open with Live Server** (recommended if you're using VS Code).

## 📌 Functionality

- Users can create multiple tasks.
- Tasks can be marked as completed after finishing them.
- Completed tasks are visually differentiated.
- Tasks can be removed whenever required.
- Changes are reflected instantly on the webpage.

## 🎯 Future Enhancements

- Task filtering (Completed / Pending).
